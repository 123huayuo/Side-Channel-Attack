# 10种情况下的应用识别实验<br>
**数据处理方式**<br>
*1 将信号频率由44100Hz降低到50Hz。*<br>
*2 将降频后的信号进行希尔伯特变换，获取信号的包络线。*<br>
*3 获取信号的包络线后进行阈值判断，判断应用可能启动的时间点。*<br>
*4 获取应用可能启动的时间点后，将该时间点后的200个点以图片的方式进行保存。*<br>
